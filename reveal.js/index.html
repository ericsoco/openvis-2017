Wall Drawing 1247

Sol LeWitt
1. algorithms as approximations of behaviors within complex systems 2. create framework/backdrop against which data can be measured: disruption in algorithmic pattern as artifact of data 3. add behavior to elements or to whole system to suggest a personality, to introduce empathetic link between data and audience

Wall Drawing 69

Sol LeWitt (Mnuchin Gallery)
Lines not long, not straight, not touching, drawn at random using four colors, uniformly dispersed with maximum density, covering the entire surface of the wall.

Process 16 (Software 3)

Casey Reas (bitforms gallery)
A rectangular surface filled with instances of Element 3, each with a different size and gray value. Draw a tiny, transparent circle at the midpoint of each Element. Increase a circle's size and opacity while its Element is touching another Element and decrease while it is not.

Substrate

Jared Tarbell
Lines grow in a particular direction until they reach the boundary of the domain, or until they collide with another line. When a line stops, at least one new line is spawned perpendicularly off of one of the existing lines at an arbitrary position. (As described by Anders Hoff)

Magnetosphere

Robert Hodgin
At its core, it is a simple physics system which plays opposing forces against each other. Some elements in the scene have an attractive force, others have a repulsive force, and over time these elements create a dynamic composition as they are pushed about by these invisible forces. randomness & probability

  • variation

  • fuzziness/uncertainty

  • probability

Wikimedia Commons
1. qualitatively indicate natural variation inherent in the data for which quantity is undefined or not useful to understanding patterns 2. represent fuzziness in data: data are observations, not always precise. think analog instruments. 3. probability to represent probability itself.

Musikalisches Würfelspiel

via Abjad documentation

Music of Changes

Craig Reynolds
Populated grid of 8x8 cells with random numbers from I-Ching; used these values to determine sounds and silences, durations, dynamics. Randomness and algorithmic composition do _not_ guarantee aesthetic pleasure.

Shepherding Randomness

Anders Hoff
Algorithms used to "shepherd" randomness Randomize velocity, keep within bounds, and sum across horizontal canvas Random walk for each point; each point randomly moves toward or away average position of neighbors Differential growth; randomly bisect edges between nodes if they are long and/or curved strongly enough
